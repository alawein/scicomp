# Docs

This lane holds SciComp's installation and API references, theory notes,
examples, and the documentation-build surface for the multi-language numerical
stack.

## Start here

- [INSTALLATION_GUIDE.md](INSTALLATION_GUIDE.md) for environment setup.
- [API_REFERENCE.md](API_REFERENCE.md) and `api/` for module and package
  surfaces.
- [examples.md](examples.md), [tutorial.md](tutorial.md), and [usage.md](usage.md)
  for guided entry points.
- [GPU_TESTING_GUIDE.md](GPU_TESTING_GUIDE.md) for accelerator-specific
  validation.
- `theory/` for the mathematical and scientific background material.
- [troubleshooting.md](troubleshooting.md) for recurring operational failures.
- [INDEX.md](INDEX.md) for the derived document map.

## Notes

- `index.rst`, `conf.py`, and `Makefile` support the docs build surface.
- Keep docs explicit about which language surface or compute backend a workflow
  targets.
